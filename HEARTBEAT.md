# HEARTBEAT.md

## Daily: Morning Reflection for Kai

- Fixed send time: 08:00 (Asia/Shanghai)
- Send one concise "晨间反思" at 08:00 each day; if missed, send at next heartbeat before 09:00.
- Format:
  1) 昨日关键结果
  2) 做对了什么 / 做错了什么
  3) 为适配 Kai 做了哪些调整
  4) 让自己更强的动作（优先免费）
  5) 今天的3个最高优先级动作
- One message per morning only; do not repeat after sending.
- Outside morning window: do not send this reflection.

## Every 4 Hours: Proactive Work Update

- Cadence: send one update every 4 hours (Asia/Shanghai), based on last sent timestamp.
- If >=4h since last update, send a concise "工作进展" message.
- Update format:
  1) 这4小时完成了什么
  2) 当前在推进什么
  3) 下一步4小时要做什么
  4) 遇到的风险/卡点（若有）
  5) 需要 Kai 决策的最小事项（若无则写“无需你介入”）
- If idle, proactively pick high-value tasks without waiting:
  - 免费能力增强（技能与流程）
  - 内容生产与转化流程优化
  - 可复用模板/SOP沉淀
- Keep it action-focused, no empty status updates.
