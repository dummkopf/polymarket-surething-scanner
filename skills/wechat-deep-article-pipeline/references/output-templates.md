# Output Templates

Use these templates to keep output stable across runs.

## A) Markdown master draft template

```markdown
# [Main Title]

> One-sentence promise

## Why this matters now

[Conclusion-first paragraph]

## Signal 1

[Conclusion-first paragraph + evidence]

## Signal 2

[Conclusion-first paragraph + evidence]

## Signal 3

[Conclusion-first paragraph + evidence]

## Structural interpretation

[What these signals mean together]

## Practical framework

[Step-by-step framework]

## 7-day action plan

- Day 1:
- Day 2:
- ...

## Risks and boundaries

[Where this may fail]

## CTA

[Specific next action]

## Sources

1. [Source title](URL)
2. [Source title](URL)
3. [Source title](URL)
```

## B) WeChat longform packaging template

```markdown
[Title]
[Subtitle]
[Lead quote]

[Body with H2/H3 sections]

[Action checklist]

[CTA + first comment seed]
```

## C) Xiaohongshu compressed version template (600–900 chars)

```markdown
[Hook sentence]

[3–5 compact points]

[Action checklist mini]

[One CTA line]

#标签A #标签B #标签C
```
