# Quality Gates

Use this checklist before handing any draft to the user.

## 1) Evidence Gate (must pass)

- [ ] At least 3 credible sources are cited.
- [ ] Every key claim has a supporting source.
- [ ] Links are accessible and not obviously broken.
- [ ] No fabricated numbers, quotes, or events.
- [ ] Claims and timestamps align with source freshness.

## 2) Structure Gate (must pass)

- [ ] Paragraph length is 2–4 lines.
- [ ] Most paragraphs start with a clear conclusion sentence.
- [ ] The argument flows from problem → evidence → implication → action.
- [ ] Includes an action checklist section.
- [ ] Includes a risk / caveat section.

## 3) Spread & Conversion Gate (must pass)

- [ ] Title includes one tension or contrarian edge.
- [ ] Opening hook creates urgency or curiosity.
- [ ] CTA is explicit and low-friction (save/comment/follow/DM).
- [ ] Includes first-comment starter and reply templates.

## 4) Voice Gate (must pass)

- [ ] Tone is sharp, calm, practical, and non-hype.
- [ ] Avoids fluffy phrasing and generic encouragement.
- [ ] Avoids overclaiming certainty.
- [ ] Maintains one consistent narrative voice.

## 5) Final Handoff Gate (must pass)

- [ ] Markdown master draft is complete.
- [ ] WeChat longform version is ready.
- [ ] Xiaohongshu compressed version is ready.
- [ ] Source list and image plan are included.
- [ ] Output package paths are clear.
